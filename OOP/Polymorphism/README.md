## Two types of polymorphism
- Compile-time Polymorphism (static polymorphism, method overloading) : Multiple methods with the same name but different parameters, resolved at compile time.
- Run-time Polymorphism (dynamic polymorphism, method overriding) : A subclass provides its own implementation of a method defined in a parent class or interface, resolved at runtime.
